## Как проверить актуальные модели

Используйте утилиту для проверки:

```bash
cd QwenApiTest
dotnet run
```

Утилита автоматически:
1. Получит список всех бесплатных моделей через OpenRouter API
2. Протестирует первые 5 моделей
3. Покажет, какие модели работают
4. Предложит рекомендуемую конфигурацию

## Рекомендуемая конфигурация

Для максимальной надежности используйте модель `openrouter/free`:

```json
{
  "AI": {
    "Provider": "OpenRouter",
    "ApiKey": "sk-or-v1-ваш-ключ",
    "Enabled": true,
    "Model": "openrouter/free",
    "BaseUrl": "https://openrouter.ai/api/v1"
  }
}
```

## Альтернативные модели

Если `openrouter/free` недоступна, попробуйте:

1. `arcee-ai/trinity-large-preview:free` - хорошее качество
2. `stepfun/step-3.5-flash:free` - быстрая работа
3. `upstage/solar-pro-3:free` - продвинутые возможности

## Ограничения бесплатных моделей

- Могут быть ограничения по количеству запросов в день
- При превышении лимита возвращается ошибка 429 (Too Many Requests)
- Некоторые модели могут быть временно недоступны
- Качество ответов может варьироваться
